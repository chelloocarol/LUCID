## 必须修改项（高风险）

### M1. 参数量声明需核实或修正

**原文（表1）**: 骨干基线参数量 **6.42 M**，LUCIDMine **6.48 M**

**实测结果**:
- `Student.pth` 对应 res_blocks=1：**2.998 M**（差距 -53%）
- res_blocks=4：**6.245 M**（差距 -3%）

**修改建议**:
- 选项A（保留6.42M声明）：在论文中明确说明使用 res_blocks=4 配置，并上传对应 Student.pth（res_blocks=4 版本）
- 选项B（更正参数量）：若实际使用 res_blocks=1，将表1中参数量更正为 ~3.0 M（骨干）和 ~3.02 M（LUCIDMine），对应增量 ~0.017 M（而非0.06 M）

### M2. FLOPs声明需核实或修正

**原文（表1）**: 骨干基线 **48.7 G**，LUCIDMine **49.6 G**

**实测结果**: 1080p 下 **88.5 G / 88.7 G**（差距约 +82%）

**修改建议**:
- 在表1注释中明确说明 FLOPs 测量时使用的输入分辨率（如"FLOPs 在 XXX×XXX 分辨率下测量"）
- 若原来在较低分辨率（如 640×1138）测量，对应约 48.7G 则合理，需补充说明
- 建议在论文中统一标注："FLOPs 在单帧 [分辨率] 输入下用 thop 库测量"

### M3. GARC公式补充residual_mask项

**原文（式11）**: `Sfinal = clip(1 + γG·S, 0.55, 1.45)`

**代码实现**:
```python
residual_mask = (1.0 - glare).clamp(0, 1) * (0.5 + 0.5 * visibility)
residual_scale = (1.0 + self.gate * learned_scale * residual_mask).clamp(0.55, 1.45)
```

**修改建议**：将式(11)补充为：
```
M(x) = (1 − G(x)) · (0.5 + 0.5V(x))
Sfinal = clip(1 + γG · S · M, 0.55, 1.45)
```
并在文字中说明 `M(x)` 为空间调制掩码，在高眩光区域（G→1）自动抑制校准强度。

## 暂不建议修改但需标注的项

### M8. 骨干基线SSIM = 0.949 vs RealAdapt实验SSIM = 0.874

仓库中的 R008 实验（student架构，RealAdapt框架）val_full_ssim 最高约 0.874，而论文中骨干基线（相同 student 架构，有监督微调）SSIM=0.949。这一差距在合理范围内（不同训练方式、不同评估集），但需要明确说明"骨干基线"是在矿山配对数据上**有监督微调后**的结果，而非 CoA 预训练权重的零样本结果。

**修改建议**：在表2注释中明确"骨干基线为户外去雾预训练+矿山数据有监督微调（不含VCA/GARC）"
